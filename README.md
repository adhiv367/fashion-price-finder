\# 👗 Fashion Price Finder



A real-time fashion price comparison app that finds the cheapest price

across Amazon, Flipkart, Myntra, Meesho, Ajio and more.



\## Features

\- Search any dress or clothing item

\- Real prices from Google Shopping

\- Compares prices across 9+ stores

\- Shows best deal and maximum savings

\- Price history saved in SQLite database



\## Tech Stack

\- Python 3.14

\- FastAPI

\- Serper API (Google Shopping)

\- SQLite

\- HTML/CSS/JavaScript



\## How to Run



1\. Install dependencies:

pip install fastapi uvicorn requests beautifulsoup4 httpx



2\. Add your Serper API key in scraper.py



3\. Run the server:

python -m uvicorn main:app --reload



4\. Open browser:

http://127.0.0.1:8000
